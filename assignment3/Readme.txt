This folder contains code in C++ to generate plastic, texture and glossy spheres along with their shadows.
To reproduce the spheres, please run the following commands.
$g++ main.cpp -lGL -lglut -lglfw -lGLEW -lGLU
$./a.out
After running these commands 3 spheres will be displayed - plastic, texture and glossy spheres from left to right in the order.
